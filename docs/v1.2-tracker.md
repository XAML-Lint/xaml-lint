# v1.2 tracker

Milestone tracker for the v1.2 release. See [`roadmap.md`](roadmap.md) for the design rationale.

Each milestone is sized to be convertible into a dedicated implementation plan when ready to execute. Update sub-checkboxes as steps complete; check the milestone heading when everything under it is done.

**Dialect references** below assume the current dialect set (WPF / WinUI 3 / UWP / MAUI / Avalonia / Uno). Preset placement (`DefaultEnabled`, `:recommended`, `:strict`) is a per-rule decision recorded in the rule's source attribute — don't forget to choose during implementation.
